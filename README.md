Wordle TUI
==========

I really just need an excuse to learn bubbletea. GIF demo at the bottom.

## Run

### Using `go install`
```sh
go install -ldflags "-s -w" github.com/sa-/wordle-tui@latest
# then you should be able to run
wordle-tui
```

### Using git clone

```sh
git clone https://github.com/sa-/wordle-tui.git
cd wordle-tui
make run
```
